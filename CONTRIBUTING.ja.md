# Contributing

## 対象範囲

このリポジトリは、`codex-spark-eclipse-legion` スキルを公開向けに整備して配布するためのものです。変更時は、ルートのスキル定義、README 群、docs サイトの説明が食い違わないようにします。

## 変更前の確認

- 挙動変更が `SKILL.md` に入るべきか、`references/` に入るべきか、あるいは両方かを確認します。
- 利用者向けの説明を更新する場合は、英語版と日本語版の構成をできるだけ揃えます。
- ビジュアル方針を変える理由がない限り、既存の SVG アセットを再利用します。

## ローカル確認

依存関係の導入:

```bash
npm install
```

作業を閉じる前に docs をビルド:

```bash
npm run docs:build
```

将来 Python 補助スクリプトを足す場合は、`python ...` ではなく `uv run ...` を使います。

## 内容同期のチェックリスト

- 運用上の挙動を変えたら `SKILL.md` を更新する
- オンボーディングや位置づけが変わったら `README.md` と `README.ja.md` を更新する
- ガイド内容が変わったら対応する英語版・日本語版 docs を両方更新する
- 例は短く、境界が明確で、担当分担が伝わる形を維持する

## コミット方針

- 小さく巻き戻しやすいコミットを優先する
- コミットメッセージから利用者影響が分かるようにする
- 可能なら docs、workflow、挙動変更を無関係に混ぜない
